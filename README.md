# IS212-G9T5

Clone the project

```bash
  git clone https://github.com/nlcchi/IS212-G9T5.git
```

Note: Please use an appropriate venv.
For MacOS, you may use

```bash
  python3 -m venv venv
  source venv/bin/activate
```

Pip install dependencies

```bash
  pip install -r requirements.txt
```

Run frontend

```bash
  cd frontend
  npm install
  npm run dev
```

Run backend

```bash
  cd backend
  python3 server.py
```
